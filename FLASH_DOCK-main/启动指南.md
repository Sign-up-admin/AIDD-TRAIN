# FlashDock 项目启动指南

## 方法一：使用启动脚本（推荐）

直接双击运行项目根目录下的 `start_flashdock.bat` 文件即可。

## 方法二：手动启动

### 步骤1：打开Anaconda Prompt或PowerShell

### 步骤2：激活虚拟环境
```bash
conda activate flash_dock
```

### 步骤3：进入项目目录（如果不在项目目录）
```bash
cd E:\Qinchaojun\FLASH_DOCK-main\FLASH_DOCK-main
```

### 步骤4：启动Streamlit应用
```bash
streamlit run FlashDock.py
```

### 步骤5：访问应用
浏览器会自动打开，如果没有自动打开，请访问：
```
http://localhost:8501
```

## 方法三：使用PowerShell一行命令

在PowerShell中运行：
```powershell
conda activate flash_dock; streamlit run FlashDock.py
```

## 注意事项

1. **确保环境已创建**：如果提示找不到环境，请先创建环境：
   ```bash
   conda create -n flash_dock python=3.10
   conda activate flash_dock
   pip install -r requirements.txt
   ```

2. **确保依赖已安装**：如果启动时出现导入错误，请安装依赖：
   ```bash
   conda activate flash_dock
   pip install -r requirements.txt
   ```

3. **模型文件**：确保已下载模型文件 `unimol_docking_v2_240517.pt` 并放置在：
   ```
   ./others/Uni-Mol/unimol_docking_v2/unimol_docking_v2_240517.pt
   ```

4. **停止服务器**：在运行服务的终端中按 `Ctrl+C` 即可停止服务器。

## 常见问题

### Q: 提示找不到conda命令
A: 请使用Anaconda Prompt而不是普通命令行，或者确保conda已添加到系统PATH。

### Q: 提示找不到streamlit命令
A: 请确保在flash_dock环境中安装了streamlit：
   ```bash
   conda activate flash_dock
   pip install streamlit
   ```

### Q: 端口被占用
A: Streamlit默认使用8501端口，如果被占用，可以使用：
   ```bash
   streamlit run FlashDock.py --server.port 8502
   ```
